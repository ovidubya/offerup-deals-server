# OfferupDeals Server

### This server should be used in conjuction with the [Android/iOS App](https://github.com/ovidubya/offerupdeals-app)

# Install

```sh
$ npm i
```

Or build a docker image

```sh
$ docker build .
```

# Run server

```sh
$ npm start
```
